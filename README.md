## 🚀 Como Executar

1. **Clone o repositório**
   ```bash
   git clone https://github.com/mantovaniandre/my-landing-page.git
   ```

2. **Navegue até o diretório**
   ```bash
   cd my-landing-page
   ```

3. **Abra no navegador**
   - Abra o arquivo `index.html` no seu navegador
   - Ou use um servidor local como Live Server (VS Code)
